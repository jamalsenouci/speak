What does the package do?

It's a tool for creating voice notifications

How does it work

It takes a text input, passes it to a text-to-speech api downloads it as a temporary file then uses the vlc command line tools to play the audio file.

Installing the package

speak depends on vlc being in the path. To install speak, type the following commands into a command line:

   pip install git+https://github.com/jamalsenouci/speak.git

Using the package interactively

  from speak import speak
  speak("hello world")

Using the package from the command line

speak("sleep 4")
